What is RAG?

Retrieval-Augmented Generation (RAG) enhances AI responses by retrieving relevant information from your own documents and using it as context for the AI model.

How Auto RAG Works

  • Index Documents - Upload and vectorize your documents using Cloudflare Vectorize
  • Query Processing - User queries are converted to embeddings
  • Semantic Search - Find the most relevant document chunks
  • Context Injection - Add retrieved context to the AI prompt
  • Enhanced Response - AI generates answers using your data

Code Example

JavaScript (Workers) 
// Query with RAG using Vectorize + Workers AI
export default {
  async fetch(request, env) {
    const { query } = await request.json();
    
    // 1. Generate embedding for the query
    const embedding = await env.AI.run(
      '@cf/baai/bge-base-en-v1.5',
      { text: query }
    );
    
    // 2. Search for similar documents
    const results = await env.VECTORIZE.query(
      embedding.data[0],
      { topK: 3 }
    );
    
    // 3. Generate response with context
    const context = results.map(r => r.metadata.text).join('\n');
    const response = await env.AI.run(
      '@cf/meta/llama-2-7b-chat-int8',
      {
        messages: [
          { role: 'system', content: `Context: ${context}` },
          { role: 'user', content: query }
        ]
      }
    );
    
    return Response.json(response);
  }
};
